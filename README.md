# demo-psicologia-clinica
SPA (Single Page Application) minimalista con tonos neutros y verde salvia. Enfocada en confianza y privacidad, incluye navegación por pestañas, acordeón clínico, contador de credenciales y panel de paciente con reserva intuitiva. Diseñada para priorizar la empatía y accesibilidad clínica.
